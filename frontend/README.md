Bienvenido a tu app Expo 
Este es un proyecto de Expo creado con create-expo-app.

empezar
instalar dependencias

bash
npm install
iniciar la app

bash
npx expo start
En la salida vas a encontrar opciones para abrir la app en:

build de desarrollo

emulador de Android

simulador de iOS

Expo Go, un sandbox limitado para probar desarrollo de apps con Expo

Podés empezar a desarrollar editando los archivos dentro del directorio app. Este proyecto usa enrutamiento basado en archivos.

obtener un proyecto limpio
Cuando estés listo, ejecutá:

bash
npm run reset-project
Este comando moverá el código inicial al directorio app-example y creará un directorio app vacío donde podés empezar a desarrollar.

aprender más
Para aprender más sobre cómo desarrollar tu proyecto con Expo, mirá los siguientes recursos:

documentación de Expo: aprendé lo fundamental o profundizá en temas avanzados con nuestras guías.

tutorial de Expo: seguí un tutorial paso a paso donde vas a crear un proyecto que corre en Android, iOS y la web.

unirte a la comunidad
Sumate a nuestra comunidad de desarrolladores que crean apps universales.

Expo en GitHub: mirá nuestra plataforma open source y contribuí.

comunidad en Discord: chateá con usuarios de Expo y hacé preguntas.
